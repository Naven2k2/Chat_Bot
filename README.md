# 🎯 AI Career Advisor Chatbot

An AI-powered career guidance chatbot built using **Streamlit** and **Google Gemini API**.

## 🚀 Features

- Provides personalized career guidance
- Suggests in-demand skills
- Creates learning roadmaps
- Offers resume and interview tips
- Restricts responses to career-related queries

## 🛠️ Tech Stack

- Python
- Streamlit
- Google Gemini API
- dotenv (for local development)
